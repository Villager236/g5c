Wenn Daten im Cache sind, lädt die Seite automatisch. Beim ersten Mal brauchst du einen Hypixel API-Key:

  2. Klicke Create API Key → Personal App → dauerhafter Key

Daten werden 10 min gecacht — danach reicht ein Klick auf ↻ Refresh.
